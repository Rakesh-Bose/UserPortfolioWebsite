Daily Routine by Rakesh

Learn javaScript

I devote 1-2 hours daily to read and learn JavaScript. I start with the fundamentals, understanding variables, data types, and functions. Then, I delve into more complex topics like DOM manipulation and asynchronous programming. I read online tutorials, JavaScript documentation, and books. I practice by working on small web projects and experimenting with code in browser developer tools. I also follow relevant blogs and forums for updates and insights, ensuring steady progress in my JavaScript.

Learn java

I allocate 1-2 hours daily to learn Java. I begin with core concepts like variables, data types, and loops, gradually progressing to object-oriented programming principles. I utilize online tutorials and Java documentation for reference. I practice coding exercises on platforms like HackerRank and Udemy courses. Additionally, I work on small Java projects to apply my knowledge practically. I seek clarification through online forums and maintain consistency in my learning to steadily improve my Java.

Learn Python

I dedicate 1-2 hours daily to learn Python. I start with basics like variables and data types, then move to control structures and functions. I practice by solving coding challenges on platforms like LeetCode and Codecademy. I review and build projects to reinforce concepts, and I seek help from online resources when needed
